class BasicCertificateChainCleaner(trustRootIndex: TrustRootIndex) : CertificateChainCleaner

A certificate chain cleaner that uses a set of trusted root certificates to build the trusted chain. This class duplicates the clean chain building performed during the TLS handshake. We prefer other mechanisms where they exist, such as with okhttp3.internal.platform.AndroidPlatform.AndroidCertificateChainCleaner.

abstract class CertificateChainCleaner

Computes the effective certificate chain from the raw array returned by Java's built in TLS APIs. Cleaning a chain returns a list of certificates where the first element is chain[0], each certificate is signed by the certificate that follows, and the last certificate is a trusted CA certificate.
