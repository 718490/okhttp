goAway

abstract fun goAway(lastGoodStreamId: Int, errorCode: ErrorCode, debugData: ByteString)

The peer tells us to stop creating streams. It is safe to replay streams with ID > lastGoodStreamId on a new connection. In- flight streams with ID <= lastGoodStreamId can only be replayed on a new connection if they are idempotent.

Parameters

lastGoodStreamId

the last stream ID the peer processed before sending this message. If lastGoodStreamId is zero, the peer processed no frames.

errorCode

reason for closing the connection.

debugData

only valid for HTTP/2; opaque debug data to send.
