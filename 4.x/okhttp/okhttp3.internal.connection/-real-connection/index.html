RealConnection

class RealConnection(val taskRunner: TaskRunner, val connectionPool: RealConnectionPool, val route: Route, rawSocket: Socket?, socket: Socket?, handshake: Handshake?, protocol: Protocol?, source: BufferedSource?, sink: BufferedSink?, pingIntervalMillis: Int, connectionListener: ConnectionListener) : Http2Connection.Listener, Connection, ExchangeCodec.Carrier

A connection to a remote web server capable of carrying 1 or more concurrent streams.

Connections are shared in a connection pool. Accesses to the connection's state must be guarded by holding a lock on the connection.
