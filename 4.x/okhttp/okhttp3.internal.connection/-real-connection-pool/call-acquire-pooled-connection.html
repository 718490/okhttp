callAcquirePooledConnection

fun callAcquirePooledConnection(doExtensiveHealthChecks: Boolean, address: Address, call: RealCall, routes: List<Route>?, requireMultiplexed: Boolean): RealConnection?

Attempts to acquire a recycled connection to address for call. Returns the connection if it was acquired, or null if no connection was acquired. The acquired connection will also be assigned to RealCall.connection.

This confirms the returned connection is healthy before returning it. If this encounters any unhealthy connections in its search, this will clean them up.

If routes is non-null these are the resolved routes (ie. IP addresses) for the connection. This is used to coalesce related domains to the same HTTP/2 connection, such as square.com and square.ca.
